# 📖 About

Programmer's Dungeon transforms learning Python into an adventure.

Instead of reading plain notes, learners climb ten dungeon floors, unlocking new programming concepts through structured lessons, practical coding exercises, boss challenges, and completion achievements.

The objective is simple:

> **Learn Python in a way that is engaging, structured, and enjoyable without sacrificing educational quality.**

---

# 🎯 Features

- 🏰 Gamified dungeon progression
- 📚 Beginner-friendly explanations
- 💻 Interactive coding exercises
- 🎯 Boss Challenges
- 📊 Beautiful markdown dashboards
- 🏆 Completion certificate
- 📈 Progress tracking
- 🎨 Website-style notebook design

# 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/Programmers-Dungeon.git
```

Install dependencies

```bash
pip install -r requirements.txt
```

Launch Jupyter

```bash
jupyter notebook
```

Open

```
Programmer's Dungeon.ipynb
```
